## Render Deployment (1 phút)
1. Đăng nhập https://render.com
2. New+ > Static Site > Connect GitHub repo `huyan189/TM-T_BT3`
3. Branch `main`, Root `/`, **Build để trống**, Publish `/` (hoặc để mặc định)
4. Deploy! Site live ngay.
